# skill-übersicht

**Erste Datei die der Agent bei jeder Aufgabe liest.**
Zeigt welche Skills existieren und wann sie gelesen werden.

---

## Pflicht-Workflow

```
1. Diese Datei lesen
2. Passenden Skill suchen → gefunden? Skill lesen, dann umsetzen.
3. Kein Skill? → skill-autor.md lesen → neuen Skill schreiben → umsetzen.
4. FORTSCHRITT.md nach Abschluss aktualisieren.
```
